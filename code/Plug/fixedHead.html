首页

说明：

引入js文件

<script src="/RedseaPlatform/skins/library/stickUp.min.js"></script>

修改 stickUp 插件的引用代码如下：

<script type="text/javascript">

 //initiating jQuery  

jQuery(function($) {

     $(document).ready( function() {

 //为 '.navbar-wrapper' class 启用stickUp插件

   $('.navbar-wrapper').stickUp({

                    parts: {

                      0:'home',

                      1:'features',

                      2: 'news',

                      3: 'installation',

                      4: 'one-pager',

                      5: 'extras',

                      6: 'wordpress',

                      7: 'contact'

                    },

                    itemClass: 'menuItem',

                    itemHover: 'active'

                  });

    });

  });

</script>


ps 上面的数字标注的是元素出现的顺序。

itemClass: 是每个导航菜单条目所对应的的class。

itemHover: 是当菜单条目处于激活状态时所被赋予的class，用于修改其外观。
