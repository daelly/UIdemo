按需载入使用说明

1、主页面加入 requreLib.js

<script src="/RedseaPlatform/skins/js/common/requreLib.js"></script>

2、在ajax加载页面里加入以下代码

<script>            
    if(typeof(requreLib)!="undefined"){
        requreLib.setplugs("icheck,select2");
    }
</script>

或

<script>            
    if(typeof(requreLib)!="undefined"){
        requreLib.setplugs("icheck,select2,http://cdnjs.cloudflare.com/ajax/libs/jquery-easing/1.3/jquery.easing.min.js,/RedseaPlatform/skins/library/rendro-easy-pie-chart/dist/jquery.easypiechart.min.js");
    }
</script>

不刷新改变URL: jquery.pjax.js

1、主页面加入 jquery.pjax.js 插件

<script type="text/javascript" src="/RedseaPlatform/skins/library/jquery.pjax.js"></script>

2、通过选择器来设置那些内容，不进行跳转页面

<script>
    $(document).pjax('a[target!=blank]', 'body');
</script>
